## 💻 Development
Platforms for coding, version control, and building software applications.

- [Android Studio](https://developer.android.com/studio) — Official IDE for Android app development.  
- [Bitbucket](https://bitbucket.org) — Git-based source code hosting and collaboration.  
- [CodePen](https://codepen.io) — Online playground for front-end experiments.  
- [Codesandbox](https://codesandbox.io) — Cloud IDE for web app prototyping.  
- [Coder](https://coder.com) — Platform for remote development environments.  
- [Eclipse](https://www.eclipse.org) — Open-source IDE for multiple languages.  
- [GitHub](https://github.com) — Code hosting with issues and pull requests.  
- [GitLab](https://gitlab.com) — All-in-one DevOps platform with CI/CD.  
- [Glitch](https://glitch.com) — Rapidly build and host web apps online.  
- [Heroku](https://www.heroku.com) — Platform-as-a-Service for deploying apps.  
- [JSFiddle](https://jsfiddle.net) — Quick browser-based prototyping environment.  
- [Katacoda](https://www.katacoda.com) — Interactive scenarios for hands-on learning.  
- [Netlify](https://www.netlify.com) — Continuous deployment for static sites and apps.  
- [PyCharm](https://www.jetbrains.com/pycharm/) — IDE specialized for Python development.  
- [Replit](https://replit.com) — Collaborative online IDE for many languages.  
- [Render](https://render.com) — Simple cloud hosting for apps and services.  
- [SourceForge](https://sourceforge.net) — Classic hub for open-source projects and downloads.  
- [SourceHut](https://sourcehut.org) — Lightweight platform for development and CI.  
- [StackBlitz](https://stackblitz.com) — Instant browser IDE for modern web stacks.  
- [Vercel](https://vercel.com) — Frontend hosting optimized for Jamstack and serverless.  
- [JetBrains Space](https://www.jetbrains.com/space/) — Integrated collaboration and CI/CD platform.

## 📊 Data & Analytics
Platforms for data engineering, visualization, and analytics.

- [Airbyte](https://airbyte.com) — Open-source connectors for data integration.  
- [Apache Airflow](https://airflow.apache.org) — Orchestrate data workflows and pipelines.  
- [Apache Hadoop](https://hadoop.apache.org) — Distributed storage and batch processing framework.  
- [Apache Spark](https://spark.apache.org) — Fast cluster computing for big data.  
- [BigQuery](https://cloud.google.com/bigquery) — Serverless analytics data warehouse by Google.  
- [ClickHouse](https://clickhouse.com) — High-performance columnar database for analytics.  
- [Databricks](https://databricks.com) — Unified analytics and data engineering platform.  
- [DBeaver](https://dbeaver.io) — Universal database client and management tool.  
- [Google Data Studio (Looker Studio)](https://lookerstudio.google.com) — Visual dashboarding and reporting tool.  
- [Jupyter Notebook](https://jupyter.org) — Interactive notebooks for analysis and experimentation.  
- [Metabase](https://www.metabase.com) — Simple open-source BI and dashboards.  
- [Microsoft Power BI](https://powerbi.microsoft.com) — Business analytics and reporting service.  
- [Mode](https://mode.com) — SQL-based analytics and collaboration environment.  
- [Observable](https://observablehq.com) — Reactive notebooks for data visualization.  
- [PostgreSQL](https://www.postgresql.org) — Advanced open-source relational database.  
- [Redash](https://redash.io) — Query, visualize, and share data dashboards.  
- [Snowflake](https://snowflake.com) — Cloud data platform for warehousing and analytics.  
- [Tableau](https://www.tableau.com) — Visual analytics and interactive dashboards.  
- [Trino](https://trino.io) — Distributed SQL query engine for analytics.  
- [Trifacta](https://www.trifacta.com) — Data wrangling and preparation tools.

## 🔒 Security & Monitoring
Platforms for security, observability, and incident response.

- [Burp Suite](https://portswigger.net/burp) — Web application security testing tool.  
- [Datadog](https://www.datadoghq.com) — Observability for logs, metrics, and traces.  
- [ELK Stack](https://www.elastic.co/what-is/elk-stack) — Logging and analytics stack (Elasticsearch, Logstash, Kibana).  
- [Grafana](https://grafana.com) — Create dashboards and visualize metrics.  
- [Nagios](https://www.nagios.org) — Monitoring and alerting for infrastructure.  
- [New Relic](https://newrelic.com) — Application performance monitoring and telemetry.  
- [OWASP ZAP](https://www.zaproxy.org) — Automated web application security scanner.  
- [PagerDuty](https://www.pagerduty.com) — Incident response and on-call orchestration.  
- [Prometheus](https://prometheus.io) — Monitoring system and time-series database.  
- [Snyk](https://snyk.io) — Find and fix vulnerabilities in dependencies.
